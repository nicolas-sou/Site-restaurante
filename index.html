Sobre o restaurante

Bistrô das Estrelas é um lugar encantador que oferece uma experiência gastronômica única. Com uma decoração elegante e aconchegante, este restaurante é o lugar perfeito para uma noite romântica ou uma celebração especial com amigos e familiares.

Sobre o espaço

O ambiente é acolhedor e intimista, com iluminação suave e música ambiente cuidadosamente selecionada, criando a atmosfera perfeita para uma noite inesquecível. Os móveis e decoração elegantes oferecem um toque sofisticado, mas ainda assim acolhedor, que faz com que os clientes se sintam em casa.

Para todas as momentos

Aniversários

O restaurante possui um espaço reservado especialmente para eventos, com capacidade para acomodar confortavelmente um grupo de pessoas. O espaço é decorado com bom gosto e equipado com tudo o que você precisa para uma celebração memorável, incluindo mesas, cadeiras, louças e talheres.

Recepcão de casamentos

O Bistrô das Estrelas é um lugar encantador para uma recepção de casamento. Com seu ambiente elegante e sofisticado, o restaurante oferece o cenário perfeito para uma celebração romântica e memorável.

Confraternização

O Bistrô das Estrelas é o lugar ideal para uma confraternização entre colegas de trabalho, amigos ou familiares. Com sua atmosfera elegante e aconchegante, o restaurante oferece o ambiente perfeito para uma celebração especial.
